# 交易历史使用指南

## 快速开始

### 查看交易历史

1. 打开钱包应用
2. 在主页点击 **"Transaction History"** 按钮
3. 等待交易加载完成
4. 浏览你的交易记录

## 功能说明

### 统计卡片

页面顶部显示交易统计：
- **Total**: 总交易数量
- **Success**: 成功的交易数量（绿色）
- **Failed**: 失败的交易数量（红色）
- **Fees**: 总手续费（SOL）

### 交易列表

每个交易显示：
- **图标**: 不同颜色表示不同类型
  - 🔴 红色向上箭头 = 发送
  - 🟢 绿色向下箭头 = 接收
  - 🔵 蓝色交换图标 = Swap
  - 🟣 紫色图片图标 = NFT 转移
- **标题**: 交易类型（Sent, Received, Swapped 等）
- **时间**: 相对时间（如 "5m ago", "2h ago"）
- **金额**: 转账金额和 Token 符号
- **状态**: Success 或 Failed

### 筛选交易

点击类型标签筛选：
- **All**: 显示所有交易
- **Sent**: 只显示发送的交易
- **Received**: 只显示接收的交易
- **Swaps**: 只显示 Swap 交易
- **NFTs**: 只显示 NFT 相关交易

### 搜索交易

在搜索框输入关键词，可以搜索：
- 交易签名（如 "abc123"）
- Token 符号（如 "USDC"）
- 地址（如 "xyz789"）
- Memo 内容

## 查看交易详情

### 打开详情页

1. 在交易列表中点击任意交易
2. 进入交易详情页

### 详情页内容

**顶部信息：**
- 交易类型标题
- 状态标识（Success/Failed）
- 金额（如果有）

**交易详情：**
- **Signature**: 交易签名（缩短显示）
- **Date**: 完整日期和时间
- **Type**: 交易类型
- **Fee**: 手续费（SOL）
- **From**: 发送者地址（如果有）
- **To**: 接收者地址（如果有）
- **Memo**: 备注信息（如果有）

### 操作按钮

**View on Solscan**
- 在 Solscan 浏览器查看
- 查看完整的链上信息
- 包括交易日志、指令等

**Copy Signature**
- 复制交易签名到剪贴板
- 用于分享或查询

**Share**
- 分享交易链接
- 通过系统分享功能发送

## 导出交易

### 导出 CSV

1. 在交易历史页面点击 **"Export CSV"** 按钮
2. 生成 CSV 格式数据
3. 包含所有交易信息

### CSV 内容

导出的 CSV 包含以下列：
- Signature（签名）
- Date（日期）
- Type（类型）
- Status（状态）
- Amount（金额）
- Token（代币）
- From（发送者）
- To（接收者）
- Fee（手续费）

### 用途

- 记账和财务管理
- 税务申报
- 交易分析
- 备份记录

## 交易类型说明

### Send（发送）
- 你发送 SOL 或 Token 给别人
- 金额显示为负数（红色）
- 显示接收者地址

### Receive（接收）
- 你接收 SOL 或 Token
- 金额显示为正数（绿色）
- 显示发送者地址

### Swap（交换）
- 通过 Jupiter 交换代币
- 显示交换的 Token
- 蓝色图标

### NFT Transfer（NFT 转移）
- 转移 NFT
- 紫色图标
- 显示 NFT 信息

### NFT Mint（NFT 铸造）
- 铸造新 NFT
- 橙色图标

### Stake/Unstake（质押/解除质押）
- 质押或解除质押 SOL
- 青色图标

### Unknown（未知）
- 其他类型交易
- 灰色图标
- 可能是复杂的 DeFi 交易

## 时间显示

### 相对时间

最近的交易显示相对时间：
- **Just now**: 刚刚（< 1 分钟）
- **5m ago**: 5 分钟前
- **2h ago**: 2 小时前
- **3d ago**: 3 天前

### 绝对时间

超过 7 天的交易显示日期：
- 格式：MM/DD/YYYY
- 例如：01/15/2024

## 刷新交易

### 下拉刷新

1. 在交易列表顶部向下拉
2. 释放后自动刷新
3. 获取最新交易

### 自动刷新

- 每次进入页面自动加载
- 显示最新的交易记录

## 常见问题

### Q: 为什么有些交易显示为 Unknown？

**A:** 可能的原因：
- 新的 DeFi 协议或程序
- 复杂的多步骤交易
- 自定义程序交易

**解决方法：**
- 点击查看详情
- 在 Solscan 查看完整信息

### Q: 交易金额为什么是 0？

**A:** 某些交易类型不涉及金额转移：
- 程序调用
- 账户创建
- 权限变更

### Q: 可以删除交易记录吗？

**A:** 不可以。交易记录来自区块链，无法删除。但你可以：
- 使用筛选隐藏某些类型
- 搜索特定交易

### Q: 交易历史可以追溯多久？

**A:**
- Solana 保留所有历史交易
- 默认加载最近 50-100 条
- 可以加载更多（未来功能）

### Q: 为什么加载很慢？

**A:** 可能的原因：
- 网络连接慢
- RPC 端点限流
- 交易数量多

**解决方法：**
- 使用付费 RPC（Helius, QuickNode）
- 减少加载数量
- 检查网络连接

### Q: 手续费为什么不同？

**A:** 手续费取决于：
- 交易复杂度
- 计算单元使用量
- 优先费用设置
- 网络拥堵情况

### Q: 可以查看其他钱包的交易吗？

**A:** 当前版本只能查看当前钱包的交易。未来可能添加：
- 查看任意地址的交易
- 多钱包切换

## 性能优化

### 加载速度

**提升方法：**
1. 使用付费 RPC 端点
2. 减少初始加载数量
3. 使用缓存（自动）

### 内存使用

**优化建议：**
- 不要一次加载太多交易
- 定期清理应用缓存
- 关闭不用的页面

## 隐私和安全

### 数据存储

- 交易数据来自区块链
- 不存储在本地
- 每次重新获取

### 隐私保护

- 地址显示为缩短格式
- 不会泄露私钥
- 所有数据公开可查

### 安全建议

- 不要分享交易签名给陌生人
- 注意钓鱼链接
- 验证交易详情
